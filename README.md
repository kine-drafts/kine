# Kine

> "Because a picture is worth 1000 tokens."

Kine is a minimalist, text-based diagramming tool. It embeds an infinite whiteboard directly into your document editor.

Type what you need, and Kine draws it using the **Gemini 3 Flash** model.

## Features

* **Flow:** Seamless text editing (Tiptap).
* **Visuals:** Hand-drawn diagram style (Excalidraw).
* **Brain:** powered by Google Gemini 3 Flash (Free Tier).
* **Speed:** <800ms generation latency.
